All examples are shown for "wzbc" station.

Now Playing widget

You can access the "/widget/now-playing" endpoint to display most recent station spins.

Available options:

  • staion - the station name
  • num - amount of spins to show. Default 1
  • sharing - whether to show sharing links. Default false
  • time - whether to show timestamps at the start of a line. Default false
  • nolinks - whether to hide links to the dj, station, music pages. Default false

Examples

No params. http://vm0.spinitron.com/widget/now-playing?station=wzbc

Num: 3. http://vm0.spinitron.com/widget/now-playing?station=wzbc&num=3

Sharing links enabled. http://vm0.spinitron.com/widget/now-playing?station=wzbc&sharing=1

Timestamps enabled. http://vm0.spinitron.com/widget/now-playing?station=wzbc&time=1

Dj/station/music links disabled. http://vm0.spinitron.com/widget/now-playing?station=wzbc&nolinks=true
