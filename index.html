Now Playing widget

You can access the "/widget/now-playing" endpoint to display most recent station spins.

Available options:

  • staion - the station name
  • num - amount of spins to show. Default 1
  • sharing - whether to show sharing links. Default false
  • time - whether to show timestamps at the start of a line. Default false
  • nolinks - whether to hide links to the dj, station, music pages. Default false

Examples

No params. https://spinitron.com/v2/widget/now-playing?station=wzbc

Num: 3. https://spinitron.com/v2/widget/now-playing?station=wzbc&num=3

Sharing links enabled. https://spinitron.com/v2/widget/now-playing?station=wzbc&sharing=1

Timestamps enabled. https://spinitron.com/v2/widget/now-playing?station=wzbc&time=1

Dj/station/music links disabled. https://spinitron.com/v2/widget/now-playing?station=wzbc&nolinks=true

Upcoming Shows widget

You can access the "/widget/upcoming-shows" endpoint to display upcoming shows.

Available options:

  • staion - the station name
  • count - amount of shows to show. Default 1

Examples

No params. https://spinitron.com/v2/widget/upcoming-shows?station=wzbc

Count: 3. https://spinitron.com/v2/widget/upcoming-shows?station=wzbc&count=3

Current Playlist widget

You can access the "/widget/current-playlist" endpoint to display current station playlist or if there is none at the moment, currently on-air show and most recent playlist will be displayed.

Available options:

  • staion - the station name
  • count - amount of shows to show. Default 1

Examples

No params. https://spinitron.com/v2/widget/current-playlist?station=wzbc
